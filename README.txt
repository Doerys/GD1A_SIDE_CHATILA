CONTROLES DU JEU 

CLAVIER :

- DEPLACEMENT DROITE - Flèche droite OU D
- DEPLACEMENT GAUCHE - Flèche gauche OU Q
- SAUT - Flèche haut OU BARRE D'ESPACE
- AGGRIPER UN MUR : MAJ
- APPUYER SUR UN BOUTON : Flèche bas OU E

MANETTE : 

- DEPLACEMENT DROITE - Flèche droite
- DEPLACEMENT GAUCHE - Flèche gauche
- SAUT - Flèche haut OU A
- AGGRIPER UN MUR : L2 OU R2
- APPUYER SUR UN BOUTON : B

Il est possible de Double jump une fois la capacité débloquée.
